# SQL Practice Repository

## Overview

This repository contains my SQL practice files and exercises created while learning SQL as part of my Data Engineering journey. The files cover fundamental to intermediate SQL concepts and are intended to strengthen my understanding of relational databases, querying, and database management.

## Topics Covered

* Database Creation
* Table Creation
* Data Types
* INSERT Statements
* SELECT Queries
* WHERE Clause
* ORDER BY
* GROUP BY
* HAVING Clause
* Aggregate Functions (`COUNT`, `SUM`, `AVG`, `MIN`, `MAX`)
* UPDATE Statements
* DELETE Statements
* ALTER TABLE
* Constraints

  * PRIMARY KEY
  * FOREIGN KEY
  * UNIQUE
  * NOT NULL
* Joins

  * INNER JOIN
  * LEFT JOIN
  * RIGHT JOIN
  * FULL JOIN
* Subqueries
* Views
* SQL Functions
* Practice Queries

## Purpose

The objective of this repository is to:

* Practice SQL regularly.

* Prepare for Data Engineering and Data Analyst interviews.

## How to Use

1. Clone the repository.
2. Open any `.sql` file in MySQL Workbench or another SQL editor.
3. Execute the queries to understand their functionality.
4. Modify and experiment with the queries to improve your SQL skills.
